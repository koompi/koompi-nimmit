# Cambodian Education Skill

Assist Cambodian schools (បឋមសិក្សា and មធ្យមសិក្សា) with teaching, administration, and parent communication. Default to everyday Khmer unless formal documents are requested.

## School System

**Grades:** ទី១–ទី៦ (Primary, ages 6-11) | ទី៧–ទី៩ (Lower secondary) | ទី១០–ទី១២ (Upper secondary)

**School year:** November → August. Khmer New Year break mid-April. National exams in August.

**Key exams:** Grade 9 (Brevet), Grade 12 (Baccalauréat)

**Core subjects:**
- Primary: ភាសាខ្មែរ, គណិតវិទ្យា, សីលធម៌, សង្គម, វិទ្យាសាស្ត្រ, កីឡា, សិល្បៈ
- Secondary: + រូបវិទ្យា, គីមីវិទ្យា, ជីវវិទ្យា, ប្រវត្តិវិទ្យា, ភូមិវិទ្យា, ភាសាអង់គ្លេស

## Lesson Plan Template

Generate in Khmer with this structure:
- មុខវិជ្ជា / ចំណាត់ថ្នាក់ / ប្រធានបទ / រយៈពេល (45 min typical)
- គោលបំណង (learning objectives, 2-3)
- សម្រង់ MoEYS standard
- សកម្មភាព: ផ្តើមមេរៀន (5 min) → បង្រៀន (25 min) → អនុវត្ត (10 min) → វាយតម្លៃ (5 min)
- ឧបករណ៍ / វិធីវាយតម្លៃ

## Student Management

**Attendance:** Track daily per class. Flag >3 consecutive or >5/month absences. Generate monthly summaries.

**Grading scale:** A ≥80% | B 60-79% | C 50-59% | D 40-49% | F <40%

**Progress reports:** Per-student with grades, attendance, teacher comments. A4 format. Include Khmer + English names when available.

## Khmer Content Generation

- Match vocabulary to grade level
- Primary: simple sentences, visual descriptions
- Secondary: academic Khmer progressively
- Math terms: បូក, ដក, គុណ, ចែក, សមីការ, អនុគមន៍, ត្រីកោណ
- Flash cards: 10-15 per set, topic-grouped
- Quizzes: mix ជ្រើសរើស, បំពេញចន្លោះ, ខ្លី

## Parent Communication

Polite everyday Khmer. Include: school name, date (ថ្ងៃទី...ខែ...ឆ្នាំ...), respectful opening, clear purpose, action required, contact info.

Common templates:
1. សុំព័ត៌មានការអវត្តរៈ
2. ចុចចំហដំណើរមកជួបគ្រូ
3. រំលឹកថ្លៃសិក្សា
4. ប្រកាសព្រឹត្តិការណ៍សាលា
5. ជូនដំណឹងវឌ្ឍនភាពសិស្ស
6. សេចក្តីជូនដំណឹងទូទៅ

## Exam Workflows

- Generate exams matching MoEYS format with point values + answer key
- Grade 9/12 prep: model on past exam patterns
- Rubric for essays: គ្រោះថ្លៃ (40%) + រចនាសម្ព័ន្ធ (20%) + ភាសា (20%) + ទម្រង់ (20%)
- Result processing: class averages, pass rates, flag students with <50% in ≥2 subjects

## Timetable

Day: 7:00-11:00 + 14:00-17:00. Each period 45 min, 5-10 min breaks. Core subjects morning. No same-subject consecutive. PE/Arts afternoon.
